# Branding Assets Needed for Skool Community

## 1. Community Logo (Profile Image)
- **Size**: 400x400px minimum (square)
- **Format**: PNG with transparent background preferred
- **Style**: Modern, tech-forward design incorporating:
  - "V" or "Vibe" element
  - Tech/coding visual elements
  - Primary color: #5865F2 (purple)
  - Secondary color: #00D9FF (cyan)

## 2. Community Banner
- **Size**: 1920x480px (4:1 ratio)
- **Format**: JPG or PNG
- **Content to include**:
  - "Vibe Marketing & Coding Academy" text
  - Tagline: "Where Automation Meets Creativity"
  - Visual elements representing:
    - AI/Automation (robot/gear icons)
    - Coding (code brackets, terminal)
    - Growth/Success (upward arrows, charts)
  - Call-to-action: "Join Free"

## 3. Design Resources

### Free Design Tools:
- **Canva**: https://www.canva.com (has Skool community templates)
- **Figma**: https://www.figma.com (for more custom designs)
- **Remove.bg**: https://www.remove.bg (for transparent backgrounds)

### Quick Solution:
1. Use Canva's "Online Community Logo" templates
2. Search for "tech community banner" templates
3. Customize with our brand colors and text

### Color Palette for Designs:
```
Primary Purple: #5865F2
Bright Cyan: #00D9FF
Accent Coral: #FF6B6B
Dark Background: #1A1A2E
Light Text: #F5F5F5
```
