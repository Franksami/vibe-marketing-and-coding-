# Website Transformation Plan - From Developer Focus to Business Focus

## Current vs New Direction

### Current Website (Too Technical)
- **Headline**: "Build $10K Client Dashboards in Days Using Cursor + Claude"
- **Focus**: Development speed, coding, technical implementation
- **Audience**: Developers who want to code faster
- **Course**: Technical modules about Cursor setup, MCP, deployment

### New Direction (Business & Money Focus)
- **Headline**: "Build a $10K/Month Business Without Coding - Using AI"
- **Focus**: Making money, business transformation, no coding required
- **Audience**: Non-technical entrepreneurs, coaches, agency owners
- **Course**: AI Transformation Business System
