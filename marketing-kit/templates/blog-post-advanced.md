## Post-Writing Optimization

### SEO Checklist
- [ ] Primary keyword in title
- [ ] Primary keyword in first 100 words
- [ ] Primary keyword in at least one H2
- [ ] Secondary keywords naturally included (3-5 times each)
- [ ] Alt text for all images
- [ ] Internal links added (3-5)
- [ ] External links to authoritative sources (2-3)
- [ ] Meta description includes primary keyword
- [ ] URL is short and keyword-rich

### Readability Checklist
- [ ] Flesch Reading Ease score: 60-70
- [ ] Average sentence length: <20 words
- [ ] Paragraphs: 2-3 sentences max
- [ ] Subheadings every 300 words
- [ ] Bullet points or numbered lists included
- [ ] Bold text for key concepts
- [ ] White space adequate

### Conversion Elements
- [ ] Lead magnet offered
- [ ] Email capture form placed
- [ ] Social share buttons added
- [ ] Related posts linked
- [ ] Comments enabled
- [ ] Author bio included

## AI Enhancement Prompts

### Title Optimization
```prompt
@content generate 10 blog title variations for an article about [TOPIC] targeting [KEYWORD]. 
Requirements:
- 60 characters or less
- Include power words
- Create curiosity
- Include the main keyword
- Use numbers where relevant
```

### Introduction Hook
```prompt
@content write 5 different introduction hooks for a blog post about [TOPIC]. 
Include:
1. A shocking statistic
2. A contrarian take
3. A personal story opener
4. A future prediction
5. A thought-provoking question
```

### Meta Description
```prompt
@seo write 3 meta descriptions for a blog post titled "[TITLE]" 
Requirements:
- 150-155 characters
- Include keyword: [KEYWORD]
- Include a benefit
- Create urgency or curiosity
- End with soft CTA
```

### Content Expansion
```prompt
@content expand this section with:
- One relevant analogy
- One specific example
- One expert perspective
- One actionable tip
[PASTE SECTION]
```

### CTA Creation
```prompt
@content write 5 different CTAs for a blog post about [TOPIC] where the goal is [CONVERSION GOAL].
Make them:
- Action-oriented
- Benefit-focused
- 2-7 words long
- Urgency-inducing where appropriate
```
